User.php : 25

Modelo que representa un Usuario registrado en IberPiso.

Un usuario puede tener el rol "usuario" (comprador/vendedor) o "admin". Extiende Authenticatable para integrarse con el sistema de login de Laravel, lo que permite usar Auth::login(), Auth::user(), etc.
